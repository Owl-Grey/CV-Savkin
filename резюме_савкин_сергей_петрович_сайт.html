Цель

Опытный системный аналитик с техническим бэкграундом в веб‑разработке и проектировании интеграций. Ищу позицию, где можно участвовать в создании и развитии сложных ИТ‑систем — от анализа бизнес‑требований до внедрения и сопровождения решений.

Ключевые компетенции

  • Системный и бизнес‑анализ полного цикла (AS‑IS / TO‑BE)
  • Проектирование требований к backend и frontend
  • UML / BPMN (включая ER‑диаграммы)
  • Формализация и поддержка ТЗ, Use Case / User Story
  • Постановка задач разработчикам/тестировщикам, приёмка
  • Проектирование интеграций REST ↔ gRPC; базовые SOAP‑подходы
  • JSON • XML • YAML • Swagger
  • SQL / MySQL; анализ БД
  • Agile/Scrum, фасилитация (дейли, грумминги, демо, ретро)
  • Работа с инцидентами, сопровождение тестирования
REST gRPC UML BPMN 2.0 Swagger JSON XML YAML SQL MySQL Jira Confluence Git Kafka (понимание)
